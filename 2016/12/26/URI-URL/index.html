URI URL URN

有个同事突然问我这个问题: url,uri有啥区别，想了一下自己晕圈了。。。

这个是个经典的技术争论，许多人都会问:URL,URI还可能有URN，它们之间的区别是什么，虽然我们简单的把URN和URL都看做是URI，但严格来说URI可以进一步划分为URL，URN或者这两者的组合，所以了解三者之间的区别将会非常有趣并让人受益匪浅。

从此图开始讲起吧

The theory

  • URI(uniform resource indentifier) identifies a resource (text document, image file, etc)

统一资源标识符（URI）提供了一个简单、可扩展的资源标识方式。URI规范中的语义和语法来源于万维网全球信息主动引入的概念，万维网从1990年起使用这种标识符数据，并被描述为“万维网中的统一资源描述符”.

  • URL(uniform resource locator) is a subset of the URIS that include a network location.

URL是URI的一种，不仅标识了Web资源，还指定了操作或者获取方式，同时指出了主要访问机制和网络位置

  • URN(uniform resource name) is a subset of URIs that include a name within a given space

URN是URI的一种，不仅标识了web资源，同特定命名空间的名字标识资源，使用URN可以在不知道其网络位置及访问方式的情况下讨论资源。
