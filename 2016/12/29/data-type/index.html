data type

JavaScript语言的每一个值，都属于某一种数据类型。JavaScript的数据类型，共六种。(es6又增添了第七中Symbol类型的值).

  • number: 整数和小数
  • string: 字符组成的文本
  • boolean: true和false两个特定值
  • null: 表示空缺，即此处应有一个值，但目前为空
  • undefined: 表示未定义或不存在，即此处目前没有任何值。
  • object: 各种值组成的集合。 通常，我们将数值，字符串，布尔型称为原始类型(primitive type)的值，即它们是最基本的类型数据类型，不能再细分了。而将对象称为合成类型(complex type)的值，因为一个对象往往是多个原始类型的值的合成，可以看作是一个存放各种值的容器。至于undefined和null，一般将它们看成两个特殊值。
    对象又可以分成三个子类型。
  • 狭义的对象object
  • 数组array
  • 函数function

狭义的对象和数组是两种不同的数据组合方式，而函数其实是处理数据的方法。JavaScript把函数当成一种数据类型，可以像其他类型数据一样，进行赋值和传递。为编程带来了很大的灵活性，体现了JavaScript作为函数是语言的本质。
这里需要明确的是，JavaScript的所有数据，都可以视为广义的对象，不仅数组和函数属于对象，就连原始类型的数据(number,string,boolean)也可以用对象引用。为了避免混淆，此后非特别声明，本教程的对象，都指狭义对象。

布尔值

布尔值表示真假两个状态。”真”用关键字true表示，”假”用关键字false表示。布尔值只用这两个值。
下列运算符会返回布尔值:

  • 两元逻辑运算符:&&(And), ||(Or)
  • 前置逻辑运算符: !(Not)
  • 相等运算符: ===, !==, ==, !=
  • 比较运算符: > < ,>=, <=

如果JavaScript预期某个位置应该是布尔值，会将该位置上现有的值自动转换为布尔值。转换规则是除了下面六个值被转为false，其他都视为true

  • undefined
  • null
  • false
  • 0
  • NaN
  • “”或’’空字符串
