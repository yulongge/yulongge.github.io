vim shortcut keys

在normal模式下

光标移动

  • h,j,k,l : 左，下，上，右
  • w: 光标移动到下一个单词首位
  • b: 光标移动到当前单词首位，如果光标已经在当前首位，就移动到前一单词首位
  • e: 光标移动到当前单词末位
  • gg： 光标移动到文本首行
  • Shift + g: 光标移动到文本尾行
  • 23 + Shift + g: 光标移动到指定的行数

    插入行

  • o: 在当前光标的下方插入新一行
  • Shift o: 在当前光标的上方插入新一行

删除与还原

  • x: 删除光标所在位置的字符
  • Shift + x: 删除光标的前一个字符
  • dd: 删除光标所在行,同时被删除的内容存于剪贴板上
  • de: 删除光标后的单词内容，同时被删除的内容存于剪贴板上
  • dw: 删除光标后的单词内容以及之后的空格，同时被删除内容存于剪贴板上
  • u: 还原上一个操作(不限于删除)

在可视模式下

剪切 与 黏贴

  • y: 复制选定块
  • yy: 复制光标所在的整行
  • d: 剪切选定块
  • dd: 剪切光标所在的整行
  • p: 黏贴文本

查找与替换

  • f + o: 在当前行的光标之后查找字母
  • f + b: 在当前行的光标之前查找字母
  • :/word : 全文查找word。按下:实际是进入了Vim的命令模式。查找操作支持正则表达式
  • r + p: 将光标之后的字符替换为字母p
  • :s/word/replace: 光标所在行的第一个word替换为replace
  • :%s/from/to/: 全文查找from并替换为to
  • :1, 50s/from/to/: 在第一行和第50行之间(含)进行搜索和替换。:45s/from/to/表示仅仅在45行进行搜索和替换。而1，$行号范围和%是等价的
  • :%s/form/to/g: 全文查找并替换为to，包含选项g的替换范围更广
  • :%s/form/to/gc: 全文查找from并替换为to，替换时询问:
    • y:表示同意当前替换
    • n: 表示不同意当前替换
    • a: 表示替换当前和后面的并且不再确认
    • q: 表示立即结束替换操作
    • l: 表示把当前的替换后结束替换操作
    • ^E向上滚屏^Y向下滚屏，用来帮助查看前后内容一决定进行操作
